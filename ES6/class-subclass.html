Class dan Subclass


By the way, jika kamu tidak familiar dengan OOP kamu bisa membaca tulisan saya tentang OOP disini.
Disana saya juga menulis tentang class dan contructor.
Class adalah salah fitur ES6 menyediakan syntax yang lebih sederhana dan rapi untuk membuat constructor sebuah object.
Berikut syntax class dan subclasss

Class
// class
class Club {
constructor(name, city) {
this.name = name;
this.city = city;
}
greet() {
return `${this.name} is a club from ${this.city}`
}
}


Subclass
Subclass adalah sebuah class yang mewarasi property atau method dari class parentnya.
Berikut contoh subclass Italia yang dibuat dari class Club sebelumnya.
// Subclass
class Italia extends Club{
constructor(name, city, country){
super(name, city);
this.country = country;
}
info() {
return `${this.name} is a football club from ${this.city}, ${this.country}`;
}
}

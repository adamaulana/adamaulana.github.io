Template Literal


Template literal/template string adalah fitur baru ES2015/ES6 yang memungkinkan kamu bekerja dengan string dengan cara yang berbeda.

Syntaxnya pun sangat sederhana
Hanya dengan menggunakan backtick.
Coba liat deh contohnya di bawah!

const greet = 'hello world'
Berikut beberap hal â€” hal yang bisa kamu lakukan dengan template literal.
Membuat multiline string / string yang lebih dari satu baris
Sebelum ES6 di perkenalkan, untuk membuat string yang lebih dari satu baris lumayan ribet sih. Sebagai contoh :

// cara membuat multiline string sebelum ES6 const string =
'baris satu\n \
baris dua\n \
baris tiga'
// atau
const string = 'baris satu\n' + 'baris dua \n' + 'baris tiga'

Ribet kan ya ???
Dengan template string membuat multiline string jadi lebih sederhana.
Perhatikan contoh di bawah!

// multiline string dengan template literal
const string = `
baris satu
baris dua
baris tiga`
Interpolation / menyisipkan Template literal juga menyediakan cara untuk menginterpolasi(menyisipkan) variable atau ekpresi kedalam string Contohnya seperti ini.

// multiline string dengan template literal
const string = `
baris satu
baris dua
baris tiga` br

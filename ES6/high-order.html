High Order Function


High order function secara sederhana adalah sebuah function yang menerima function lain sebagai argumen atau mengembalikan function sebagai output.

Berikut 4 high order function yang paling sering digunakan.
Agar lebih mudah memahami penggunaan method — method ini, saya sarankan langsung praktek ya.
Bisa menggunakan online IDE seperti repl

Map
Method map mereturn/mengembalikan array baru dari hasil memanggil fungsi untuk setiap elemen array.
Methode map memanggil fungsi yang disediakan satu kali untuk setiap elemen dalam sebuah array secara berurutan.
Map tidak menjalankan fungsi untuk elemen array tanpa nilai.
Method map tidak akan mengubah original array.
let animes = [
{ id: 1, name: 'Saint seiya'},
{ id: 2, name: 'Naruto'},
{ id: 3,name: 'Nanatsu no Tazai'}
]
animes.map(anime => anime.name)

Filter
Method filter membuat array baru yang elemennya diisi dari semua elemen array yang memenuhi kondisi.
Method filter tidak menjalankan fungsi untuk elemen array tanpa nilai.
Method filter tidak mengubah array asli.
Dari contoh diatas kita mencoba menfilter original array animes yang itemnya punya properti name = Naruto.
Outputnya adalah sebuah array baru dengan satu item.

Find
Methode find mengembalikan nilai elemen pertama dalam array yang sesuai dengan kondisi yang diberikan.
Methode find mengeksekusi fungsi satu kali untuk setiap elemen yang ada dalam array.
Jika menemukan elemen yang di cari , find () mengembalikan nilai elemen array tersebut (dan tidak memeriksa nilai yang tersisa).
Jika tidak menemukan elemen yang dicari, maka akan mengembalikan nilai undefined.
let animes = [
{ id: 1, name: 'Saint seiya'},
{ id: 2, name: 'Naruto'},
{ id: 3,name: 'Nanatsu no Tazai'}
]
animes.find(anime => anime.name === "Naruto")

Reduce Secara literal reduce berarti mengurangi.
Methode reduce digunakan untuk mereduce elemen — elemen array ke sebuah nilai tunggal.
Methode reduce mengeksekusi fungsi yang disediakan untuk setiap elemen dalam array (dari kiri ke kanan).
Nilai return dari fungsi disimpan dalam akumulator (hasil / total).
Metode reduce akan menerima sebuah callback function.
Callback function ini harus menerima 2 paramerter wajib yaitu accumulator dan currentValue.
Btw, accumulator dan currentValue dapat kamu ganti dengan nama lain.
Sebagai contoh kita akan mencoba mereduce array numbers dengan menguranginya dengan angka setelahnya.
Perhatikan contoh kodenya di bawah!
let numbers = [175, 50, 25]
const result = numbers.reduce((accumulator, currentValue) => accumulator - currentValue)
// result = 100 Atau contoh yang sedikit lebih komplek.
Kita akan menjumlah harga setiap barang yang ada dalam array fruits.
const fruits = [
{name: "Pisang", price: 2000},
{name: "Alpukat", price: 3000},
{name: "Melon", price: 5000}
]
// tanpa destructuring
const total = fruits.reduce((totalPrice, currentItem) => totalPrice+currentItem.price,0)
// dengan destructuring
const total2 = fruits.reduce((totalPrice, {price}) => totalPrice + price, 0)
console.log(total) // 10000
console.log(total2) // 10000
Intinya method reduce akan mengembalikan single value.

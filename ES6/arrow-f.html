Arrow Function


Function adalah salah satu hal yang paling esensi dalam dunia pemprograman.
ES6 memperkenalnya sebuah standar baru dalam membuat sebuah fungsi.
Arrow function akan membuat kode lebih ringkas
Selain itu, juga untuk menyerderhanakan ruang lingkup fungsi itu sendiri dan penggunaan this keyword.
Berikut syntax dan beberapa aturan arrow function.

//function tanpa parameter dan body function hanya 1 baris
const nameFunction = () => console.log('Hello world');
//function dengan 1 parameter dan body function hanya 1 baris
const nameFunction = name => console.log(name);
//function dengan lebih dari 1 parameter dan body function hanya 1 baris
const nameFunction = (name, age) => console.log(name & age);
// function dengan body lebih dari 1 baaris
const nameFunction = name => {
console.log('Hello world');
console.log('Hello world again')
}

Spread operator
Spread operator digunakan memperluas data yang iterable(array / objek) menjadi element individual.
arr = [1, 2, 3, 4]
arr2 = [...arr, 5, 6, 7]
console.log(arr2); // [1,2,3,4,5,6,7]
let club = { id: 1, country: 'Italia'}
let milan = {...club, name: 'AC Milan'}
console.log(milan)
// { id: 1, country: 'Itialia, name: 'AC Milan'}

Rest parameter
Rest parameter berfungsi untuk mengumpulkan sisa daftar argumen dari sebuah function ke dalam sebuah array
function test(a, b, ...rest){
console.log(...rest)
}
test(1,3,4,5,6)
// 1 adalah argumen untuk parameter a
// 3 adalah argumen untuk parameter b
// 4,5,6 adalah sisa paremeternya yang kemudian akan di satukan dalam sebuah array
//outputnya [4,5,6]
